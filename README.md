# Sanjo Buchhaltung

Eine Buchhaltungssoftware für Deutschland. Noch in früher Entwicklung.

## Geplante Features

* Doppelte Buchführung
* Jahresabschluss
* E-Bilanz
* Umsatzsteuer-Voranmeldung und Umsatzsteuererklärung
